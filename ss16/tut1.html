Bewertung

Grundanforderung
  • 1 Hausübungsbeispiel (unbewertet)
  • 1 Gruppenbeispiel (3er Gruppen, min. 8 von 16 Punkte)
  • Klausur-Basisteil (min. 3 von 6 Punkte)
  • Die Beispiele müssen rechtzeitig abgegeben werden
    (erkennbarer Versuch, keine Plagiate)
  • Bewertet wird Programmierstil, Struktur, Korrektheit und Funktionalität
  • Mitarbeit wird durch ein Abgabegespräch überprüft
  • Das Gruppenbeispiel kann beim 2. Abgabegespräch noch einmal verbessert werden.
Erweiterte Anforderungen
  • 1 erweitertes Gruppenbeispiel (max. 16 Punkte)
  • 1 erweiterter Klausurteil (max. 4 Punkte)
  • Mitarbeit wird durch ein Abgabegespräch überprüft

Objektorientierte Programmierung

  • Eine Klasse vereint Daten (Strukturen) und Funktionalität (Funktionen).
  • Entspricht oft Objekten in der reellen Welt (Spieler, Kamera, etc.)
  • Vereinfachte Schnittstelle (interne Details können verborgen werden)
// Ball.h
#ifndef BALL_H_INCLUDED
#define BALL_H_INCLUDED
class Ball
{
private:
  std::string color_;
  double x_, y_;

public:
  Ball(std::string color);
  void move(double dx, double dy);

  std::string getColor();
  void setColor(std::string new_color);
};
#endif

Ball.cpp

  • Trennung Implementierung und Definition.
// Ball.cpp
#include "Ball.h"
Ball::Ball(std::string color) : color_(color), x_(0), y_(0)
{
}
void Ball::move(double dx, double dy)
{
  x_ += dx;
  y_ += dy;
}
std::string Ball::getColor()
{
  return color_;
}
void Ball::setColor(std::string new_color)
{
  color_ = new_color;
}

Getter / Setter

private Attribute, öffentliche Getter und Setter

  • neue Werte können validiert werden
  • abhängige Daten können automatisch aktualisiert / angepasst werden
  • Werte können im Getter berechnet werden
  • interne Datenformat kann später geändert werden, die öffentliche API bleibt gleich
std::string Ball::getColor()
{
  return color_;
}
void Ball::setColor(std::string new_color)
{
  color_ = new_color;
}

Methoden

Eine Funktion innerhalb einer Klasse heißt Methode. Methoden werden üblicherweise mit Verben benannt.

void Ball::move(double dx, double dy)
{
  x_ += dx;
  y_ += dy;
}

Strings

  • Repräsentieren Text bzw. Zeichenketten
  • C++ Variante von char*
  • Bietet sehr viele komfortable Funktionen zur Bearbeitung von Text
#include <string>
#include <iostream>

int main()
{
  std::string str("Hello World!");

  std::cout << str << std::endl;

  return 0;
}
Ausgabe: 
Hello World!

Strings: Zugriff auf Zeichen mit "[ ]" oder string::at

#include <string>
#include <iostream>

int main()
{
  std::string str("Hello World!");

  std::cout << "Second character of str: " << str[1] << std::endl;

  std::cout << "Seventh character of str: " << str.at(6) << std::endl;

  str[1] = "Y";

  std::cout << "Second character of str: " << str[1] << std::endl;

  return 0;
}
Ausgabe: 
Second character of str: e
Seventh character of str: W
Second character of str: Y

Strings: Vergleichen mit string::compare

#include <string>
#include <iostream>

int main()
{
  std::string str("Hello World!");
  std::string strSEP("SEP> ");

  if(str.compare(strSEP) == 0)
  {
    //strings are equal
    std::cout << str << " == " << strSEP << std::endl;
  }
  else
  {
    //strings are not equal
    std::cout << str << " != " << strSEP << std::endl;
  }

  return 0;
}
Ausgabe: 
Hello World! != SEP>

Strings: Umwandlung von Zahlen zu Strings mit std::to_string

#include <string>
#include <iostream>

int main()
{
  std::string pi_str = "pi is ";
  pi_str += std::to_string(3.1415926);

  std::cout << pi_str << std::endl;

  int forty_two = 42;

  std::string str = "answer to life the universe and everything is ";

  str += std::to_string(forty_two);

  std::cout << str << std::endl;

  return 0;
}
Ausgabe: 
pi is 3.141593
answer to life the universe and everything is 42

Strings: Verwendung als Klassenmember

#include <string>
#include <iostream>
#include "Ball.h"

int main()
{
  Ball yellow_ball("Yellow");
  Ball another_yellow_ball();

  std::string yellow_color("Yellow");

  another_yellow_ball.setColor(yellow_color);

  std::string yellow_ball_color = yellow_ball.getColor();

  if(yellow_ball_color == another_yellow_ball.getColor())
  {
    std::cout << "Both balls are yellow!" << std::endl;
  }

  return 0;
}
